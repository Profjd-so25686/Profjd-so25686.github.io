Cyber Security Module Overview

Cyber Security has become a global issue affecting society at all levels. The increasing reliance on the Internet, World Wide Web, Internet of Things (IoT), and cloud-based critical services have made Cyber Security a critical issue for institutions and governments globally.

Module Objectives

This module is designed to meet the growing demand for Cyber Security professionals. It will:

  • Explore security implications of interconnected online services
  • Cover concepts and principles in Cyber Security
  • Discuss governance and ethical frameworks
  • Equip students with theoretical and practical skills
  • Raise awareness of ethical and professional responsibilities

Key Topics

  • Professional roles, skillsets, and ethical responsibilities in Cyber Security
  • Relevant computing architectures, practices, applications, and standards
  • Core concepts: cryptography, penetration testing, and ethical hacking
  • Programming concepts for security-based decisions
  • Secure information system implementation using object-oriented approaches
  • Ethical, social, and corporate responsibilities in Cyber Security

Learning Outcomes

Upon completion, students will be able to:

  • Identify and explain basic computer system components, considering Cyber Security issues
  • Evaluate key vulnerabilities and threats in software and network security
  • Apply concepts of secure object-oriented design to business security decisions
  • Critically evaluate solutions for security issues

This module provides a high-level introduction to topics covered in greater detail throughout the Master's programme and in Cyber Security careers.
